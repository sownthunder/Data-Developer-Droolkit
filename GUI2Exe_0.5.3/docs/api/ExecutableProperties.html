class ExecutableProperties.ExecutableProperties(parent)¶

A small panel in the lower left corner of the main GUI, which holds information about the executable size/executable folder size and number of files generated in the building process.

__init__(parent)¶

Default class constructor.

Parameters:

  • parent: the parent widget.
BuildImageList()¶
Build the image list for the list control.
DoLayout()¶
Layouts the widgets in the panel.
InsertColumns()¶
Inserts the columns in the list control.
PopulateList(blank=False, project=None)¶

Adds executable information to the listCtrl.

Parameters:

  • blank: whether to clear the list control or not;
  • project: the project from where to get all the data.
